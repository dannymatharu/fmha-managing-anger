Club Behaviour Implementation Guide

Four Pillars. One Honest Audit.

The Four Pillars from Module 5, turned into something your committee can actually work through. Be honest about what's in place versus what you assume is in place, then commit to one action per pillar.

How to use this: Go through each pillar with your committee. Answer the question straight, then take the first action before your next meeting — not eventually, before the next meeting.

The four pillars
1

Clear Expectations

Is there a written code of conduct that every new player, parent and volunteer actually receives — not one that just exists somewhere on a shared drive?
First action

Adopt or adapt the Code of Conduct tile and confirm who's responsible for handing it to every new joiner.

2

Visible Leadership

Do committee members and senior coaches visibly model the standard, or is it only enforced downward onto players and parents?
First action

Name one committee member who owns behaviour standards publicly — not administratively, publicly.

3

Consistent Consequences

Are sanctions applied the same way regardless of who's involved — does the star player's parent get the same response as anyone else?
First action

Agree a simple three-strike written process before an incident happens, not while one is unfolding.

4

Proactive Communication

Does the club say anything about behaviour before the season starts, or only react once a referee report lands?
First action

Send one pre-season email or run one meeting where the standard gets stated out loud.
